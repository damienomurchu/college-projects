### What is this repository ? ###

This repository is the completed code submission for the 1st assignment of the 2016 Programming module of the Higher Diploma in Science, Computer Science programme.

The assignment spans 3 Java coding questions, comprised of:

- Sphere; implement a sphere class 
- DemoArray; implement a variety of array operations 
- Library; implement a book lending program
- Strings; perform a range of string manipulations
- Clock; implement added functionality in a clock gui app

Submitted: 2nd April, 2016
